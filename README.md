# Program
Easily set up a Church of Jesus Christ of Latter-day Saints Sunday Service Program for your ward, including a QR code generator.

# Implementation Steps:
1. Create a copy of this google sheet template into your own google drive: (https://docs.google.com/spreadsheets/d/1-85YBuMWFVscnSn8Ie7LfuxC0k6Eeh0uygFO1dlQYwQ/edit?usp=sharing). This will serve as the backend sheet you will populate that the website will be getting inputs from.
